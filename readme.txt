Noted PRO allows you to create categories, notes and writing prompt that can be placed on any page or post using a shortcode.

== Description ==

Bloom Factor's Noted PRO allows an author or instructor to deliver writing prompts and gives users the ability to take notes on courses, lessons, topics or assignments, on any page or post.

Members are able to answer one or multiple questions that are added from the backend by the admin. A User is able to rearrange the notes as they want and all answers can be saved. Noted PRO is 100% mobile responsive, easy to use and adds significant value to course material and reader retention.

In the free version, you are able to add categories and 2 notes per page or post. The PRO version allows you to add unlimited categories and notes to any page or post.

== Installation ==

Upload the zip file using the Wordpress Plugins >Add New or
Upload the entire folder to the /wp-content/plugins/ directory.
Activate the plugin through the ‘Plugins’ menu in WordPress.

On the left menu, there will be the Noted to add new categories, add a new note and a way to generate the shortcode required to view the category or categories on a page or post. (Noted > Shortcode Generator).

Configure the shortcode by selecting a category you would like to user or you can display all notes instead as well.

== Frequently Asked Questions ==

1. Who would use Noted PRO?
Noted PRO is perfect for instructors building online courseware or writers working on character and plot development.
2. Does Noted PRO work with any Wordpress theme? What is the minimum Wordpress version? Yes, Noted Pro works with any Wordpress theme, it requires at least Wordpress 4.8 or above.
3. Does Noted PRO work with Gutenberg? Yes, it does. Add the shortcode via the widget > shortcode block.
4. Can Noted PRO be used on a Wordpress Multisite? Yes, But each install requires a license! Licenses are available for purchase from within the Wordpress dashboard.
5. What are three benefits of Noted PRO? You can have multiple notes on a page or post. The free version allows two notes on any page or posts the paid version allows as many as you want.
6. Can you have different notes on different pages and posts? Yes. You can use categories to order which notes you want on different posts or pages.
7. How do you add the notes to a post or page? After you have created the note or notes, you use the shortcode generator to create the correct shortcode.


= What is the shortcode? =

You can generate shortcode from backend by going to (Noted > Shortcode Generator). Here is example : [Noted cat="57" ] in this example 57 is the note tag id.

= How to add Noted? =

You can add a new Note by going to Noted > Add New and from that point you can add questions to that note from the next section.

= What are the plugin requirements? =

This plugin can be used with any well-crafted Standards compliant Wordpress theme, no specific theme requirement for this plugin is required.
